## Key Concepts Reference

### OSI Model
| Layer | Name | PDU | Device | Example |
|-------|------|-----|--------|---------|
| 7 | Application | Data | — | HTTP, DNS |
| 6 | Presentation | Data | — | SSL, TLS |
| 5 | Session | Data | — | NetBIOS |
| 4 | Transport | Segment | — | TCP, UDP |
| 3 | Network | Packet | Router | IP |
| 2 | Data Link | Frame | Switch | MAC |
| 1 | Physical | Bits | Hub | Cable |

### Key Port Numbers
| Port | Protocol | TCP/UDP | Purpose |
|------|----------|---------|---------|
| 22 | SSH | TCP | Secure remote access |
| 23 | Telnet | TCP | Insecure remote access |
| 25 | SMTP | TCP | Send email |
| 53 | DNS | UDP | Domain resolution |
| 80 | HTTP | TCP | Web unencrypted |
| 443 | HTTPS | TCP | Web encrypted |
| 3389 | RDP | TCP | Remote desktop |
